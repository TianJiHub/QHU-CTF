# QHU-CTF竞赛平台

## 项目概述

本项目采用前后端分离架构，构建现代化的CTF（Capture The Flag）竞赛平台。技术选型注重性能、安全性和可维护性，支持高并发访问和实时数据更新。

## 版本兼容性

本项目严格遵循语义化版本控制，确保：
- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正

所有依赖库版本均经过充分测试，确保稳定性和安全性。定期更新依赖版本，及时修复安全漏洞。
